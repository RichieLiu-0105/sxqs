Python基础数据类型

一、Numbers（数字）

支持int、float、bool、complex（复数）、bool（布尔）

内置type()函数可以用来查询变量所指的对象类型，或者用isinstnce()来判断

数值类型是不允许改变的，如果改变数字数据类型的值，将重新分配内存空间。

二、String（字符串）

字符串用单引号（’’）或双引号（””）括起来，同时使用反斜杠（\）转义特殊字符

两种索引方式：从左往右，从右往左

可以切片，获取一段字串。用冒号分隔两个索引，形式为变量【头下标：尾下标】

字符串不可变

三、List（列表）

列表是写在方括号之间，用逗号分隔开的元素列表。

支持索引和切片。

列表是可变的

四、Tuple（元组）

元组是写在小括号里，元素之间用逗号隔开

支持索引和切片

元组不可变的

五、Sets（集合）

集合是一个无序不重复元素的集

可以使用大括号或者set()函数创建set集合，注意：创建一个空集合必须用set()而不是{}，因为{}是用来创建一个空字典

六、Dictionary（字典）

字典是一种映射类型，它是一个无序的键：值对集合

关键字必须使用不可变类型，也就是说list和包含可变类型的tuple不能做关键字

在同一个字典中，关键字还必须互不相同

够造函数dict()直接从键值对sequence（序列）中够造字典

七、总结数据类型

数据类型：

  1. 字符串string
  2. 数字类型Number
    1. 整型int
    2. 浮点float
    3. 复数complex
    4. 布尔bool
  3. 列表list
  4. 元组tuple
  5. 字典dict
  6. 集合set

特征：

  1. 可变数据类型：列表、字典、集合
  2. 不可变数据类型：字符串、数字、元组
  3. 容器类型数据：字符串、列表、元组、集合、字典
  4. 非容器类数据：数字、布尔类型

附:进制转换

进制：

  1. 二进制
  2. 八进制
  3. 十进制
  4. 十六进制

转换：

  1. 十进制转换为二进制（除2取余，逆序排列）
  2. 二进制转换为十进制（按权展开求和）
