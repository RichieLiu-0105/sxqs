效果监测及策略修改

监测自身数据及竞争对手数据(排名、流量、收录、外部链接等)。

SEO计划和实施是否真的符合搜素引擎质量规范和算法规律，谁也不能事先100%确定，只能以最后效果为依据。

一、SEO完整过程

  1. 竞争研究：关键词研究和竞争对手
  2. SEO计划：诊断网站，找出不足，提出优化方案及外部链接建设方案
  3. 网站优化：结构调整及页面优化
  4. 外部链接建设：与网站优化同时进行
  5. 效果监测及流量分析：数据统计分析
  6. 策略修改：基于监测数据做修正

二、目标设定及测量

目标要明确，具有可操作性。网站目标就是你想让用户在网站做什么，最想让用户做的具体事就是你的目标。目标要具有

  1. 现实性
  2. 可测量
  3. 行动性
  4. 明确单一

三、数据监测

SEO监测的数据分为非流量监测和流量监测两个部分：

  1. 非流量监测

    1. 收录数据（收录是网站排名和流量的基础）

      1. 总收录数：
      2. 特征页面收录：
      3. 各分类收录数：
      4. 收录不充分原因：
        1. 权重不够高
        2. 结构有问题
        3. 内部链接分布不均匀
        4. 搜索引擎不友好
        5. 网站内容原创度不够

    2. 排名监测（关键词排名）

      1. 首页目标关键词
      2. 分类页面目标关键词
      3. 文章页面关键词

    3. 外部链接数据

      1. 首页总链接数
      2. 网站所有页面的总链接数
      3. 连向网站的总域名数
      4. 特征页面链接数

    4. 转化和销售

      无论收录、排名和流量，最红都为了转化。与流量来路（搜索流量、输入流量、其他网站点击）以及搜索流量中哪个关键词对应起来。这样，什么样的流量价值最高，什么关键词价值最高，通过网站流量分析一目了然。

  2. 流量数据监测（要借助流量分析软件）

    1. 读原始日志文件
    2. 常用流量统计和分析工具
      1. 基于在页面上插入统计代码（简单易用）
        1. Google Analytics
          1. 访问数
          2. 绝对唯一访问者数
          3. 页面访问数（PV）
          4. 平均页面访问数（黏度）
          5. 跳出率（是否满足需求）
          6. 流量来源（直接访问、点击流量、搜索流量）
          7. 最常见进入页面（landing page）
          8. 转化
        2. 百度统计
      2. 基于对原始日志文件进行分析
        1. AWStats

四、策略改进

  1. 收录是否充分
  2. 哪些页面带来搜索流量
  3. 目标URL排名如何
  4. 挖掘关键词
  5. 其他搜索引擎流量
  6. 长尾效果
  7. 关键词排名下降
  8. 链接诱饵成效
  9. 发现链接伙伴
  10. 寻找潜力关键词
