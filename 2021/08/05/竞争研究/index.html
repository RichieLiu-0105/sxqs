竞争研究

在做任何事情时，不能脑袋一拍贸然进入某个领域，除了运气，否则很难成功。

对于搜索营销来说，入局某领域时，需要先做竞争研究，了解市场竞争对手情况，最重要是规划好目标关键词，如果没有规划，那么会有两个结果：

  1. 想做的关键词排名上不去
  2. 认为不错的关键词排名到第一也没什么流量

网站就是产品介绍，如同写作文，要有好的框架，优质的内容，主次结构清晰，最重要是定位好关键词，因为用户搜索时，最重要就是关键词匹配，如果目标关键词没有设定好，内容再相关，搜索引擎也很难让你上位。

一、竞争对手研究

包括关键词研究，竞争对手研究及现有网站评估诊断，其中关键词研究最重要，意义包含：

  1. 确保目标关键词有人搜索（有搜索流量行为）：核心关键词不能想当然，确认用户搜索次数有一定量级。
  2. 降低优化难度：找出搜索次数比较多，同时难度不太大的关键词，优化才能在一定的预算、周期下取得效果。热门关键词你扛不住
  3. 寻找有效流量：排名和流量都不是目的，有效流量带来的转化才是目的。流量本身不一定是资产，只有能转化的有效流量才是资产。
  4. 搜索多样性：长尾流量
  5. 发现新机会：关键词扩展工具（挖掘相关关键词，用户搜索的词）、分析流量，挖掘新财富大陆

二、关键词选择

选择恰当正确的关键词，才能让网站走在正确的大方向上。确定什么样的关键词决定了网站内容规划、链接结构、外部链接建设等重要后续步骤。选择关键词原则：

  1. 内容相关：目标关键词必须与网站内容有相关性。（转化型-产品，需要考虑有效流量；流量型-广告，只需要考虑流量）
  2. 搜索次数多，竞争小：关键词挖掘、扩展、列出搜索次数（关键词工具及指数都提供搜索次数数据）及竞争程度数据（需要参考数据比较多，不确定性，比较复杂），还是可以找到合适效能的关键词，相同投入下，关键词效能高的词获得好排名的可能性较高，可以带来更多流量。关键词竞争判断参考：
    1. 搜索结果数：搜索结果页面数量
    2. intitle结果数：标题出现关键词的页面才是真正竞争对手
    3. 竞价结果数（广告数）：：广告数（关键词研究和广告投放），竞价数真金白银与你竞争的真实存在的竞争对手数目
    4. 竞价价格（广告点击价格）：价格越高，竞争越激烈
    5. 竞争对手：自然结果排在前面的主要竞争对手情况，包括外部链接数量、网站结构、页面关键词优化等
    6. 内页排名数量：首页还是内页，首页是权重最高的页面
  3. 主关键词不可太宽泛：关键词宽泛，竞争大，所花代价高，搜索词意图不明显，转化率也低。
  4. 主关键词也不可太特殊：搜索次数低、没人搜索。
  5. 商业价值：搜索词研究，判断搜索词商业价值，购买意图强的关键词应该是优化时最先考虑的，无论内容规划、还是内部链接安排，都要予以侧重

三、核心关键词

核心关键词通常就是网站首页的目标关键词。整个网站会有很多目标关键词，不可能都集中在首页进行优化，要合理分布在整个网站，形成金字塔结构。

  1. 难度最高、搜索次数最多的两、三个核心关键词，放在首页；
  2. 难度次一级、数量更多的关键词，放在栏目首或分类首页；
  3. 难度更低的关键词，数量更庞大，放在具体文章页面。

整个网站的关键词按照搜索次数、竞争程度、优化难度逐级分布。关键在于确定核心关键词，首页核心关键词一旦确定，其下的栏目及产品页面关键词也就相应确定了。

  1. 头脑风暴：用户会用什么搜索词
  2. 朋友同事：用户搜什么关键词
  3. 竞争对手：网站首页源文件标签、正文内容
  4. 搜索次数：工具（关键词工具或指数）
  5. 核心关键词：两三个比较合适，根据实力选择合适正确有效能的关键词

四、关键词扩展

需要寻找比核心关键词搜索次数少一些的更多关键词，安排在分类或频道首页。扩展方式：

  1. 关键词工具：关键词工具或指数等工具来拓展相关关键词
  2. 搜索建议：下拉搜索
  3. 相关搜索：相关搜索
  4. 其他关键词拓展工具：其他关键词工具
  5. 各种形式的变体：同义词、相关词、简写、错字
  6. 补充说明文字：地名、品牌、限定和形容词
  7. 网站流量分析：用户使用什么关键词来到网站
  8. 单词交叉组合：交叉组合词

五、关键词分布

经过核心关键词确定和关键扩展后，应该得到相关关键词列表，这些词需要合理分布在网站上。

  1. 金字塔结构：核心关键词位于塔尖，只有两三个，使用首页优化；次一级关键词相当于塔身部分，可能有几十个，放在一级分类首页；再次一级放置于二级分类首页，可能有几百上千个；更多长尾关键词处于塔底，放在页面。
  2. 关键词分组：有逻辑的分组，每一组关键词针对一个分类
  3. 关键词布局：
    1. 每个页面只针对两三个关键词，不能过多，写作时有针对性，页面突出主题
    2. 避免内部竞争，每个页面针对的两三个关键词，不要重复出现在多个页面上
    3. 关键词研究决定内容策划：关键词决定内容策划撰写
    4. 关键词-URL对应表：关键词关联目标URL

六、长尾关键词

较长、比较具体的、搜索次数比较低的词就是长尾关键词。

七、三类关键词

按照搜索目的不同，关键词分为三种：导航类、交易类、信息类

  1. 导航类：用户知道要访问哪个网站。截流（增加外部链接和锚文字）
  2. 交易类：商业价值最大，直接、精准、购买
  3. 信息类：长尾页面，进入视野

八、预估流量及价值

关键词研究的最后一步时预估流量及价值。

  1. 确定目标排名

  2. 预估流量：关键词搜索次数、关键词预计排名、搜索结果页面各排位的点击率（不准确）

    1. 搜索次数
    2. 点击率
    3. 其他原因
      1. 整合搜索
      2. 多次点击
      3. 关键词类型
      4. PPC影响自然排名点击
      5. 季节性波动

  3. 预估搜索流量价值：流量不是目标，订单和盈利才是目标。通过预计流量，结合网站转化率及平均订单销售额和平均每单毛利，就可以计算出预期通过SEO获得的搜索流量能带来的实际价值

九、关键词趋势波动和预测

  1. 长期趋势
  2. 季节性波动
  3. 社会热点预测

十、竞争对手研究

  1. 域名权重相关数据
    1. 域名年龄
    2. PR值
    3. 快照日期
    4. 收录页面总数
    5. 外部链接情况
    6. 主要目录收录情况（Hao123等）
    7. 社会化媒体出现情况
  2. 网站优化情况
    1. 页面标题是否包含关键词
    2. 网站栏目是否清晰合理
    3. URL是否静态化
    4. 网站链接结构是否合理有效
    5. 内页距离首页距离
    6. 主要页面是否有实质内容
    7. 导航系统是否使用Flash等不利于爬行的方法
    8. 页面是否使用了H1标签等
  3. 网站流量：
    1. 精准流量不可见，可利用工具辅助

十一、快速网站诊断

  1. 第一步：竞争对手指标，计算页面收录比例
  2. 第二步：网站管理员（站长）工具
    1. robots文件检查
    2. 首选域设置
    3. 关键词排名
    4. 外部链接
    5. 网站内容
    6. 内部链接
    7. 抓取错误及统计
    8. HTML建议
    9. 模拟蜘蛛抓取
    10. 网站性能

十二、竞争

搜索营销只是市场工作一部分，SEO又是搜索营销工作的一部分，SEO的载体是网站，只为让网站获得流量和排名，终极目的是流量转化成金钱。

但优化的前期条件是你需要定位好你是谁，你的价值，你的客户群以及搜索引擎规则和你的用户都是怎样的搜索行为，你需要从他们行为中提炼出关键词并优化，这样才能让他们在搜索时看到你。

你同样需要注意你市场中的竞争对手，要了解他们的状况，以便让你更好的设定目标及制定可执行的战术。
