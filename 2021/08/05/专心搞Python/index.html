专心搞Python

不会编程在现在这个社会是真的不行，效率大大打折扣，所以学会编程还是必须要尝试的，用编程来替代重复劳动。

Python入门

  1. 查看Python版本

    python -v

  2. 第一个Python程序

    #!/usr/bin/python3：指定用什么解释器运行脚本以及解释器所在的位置，如果解释器没有装在/usr/bin目录，改成其所在目录就行了，或者更通用的方法是：#!/usr/bin/env python

  3. 安装运行Python

    1. 交互式解释器
    2. 命令行脚本
    3. 集成开发环境（IDE）

  4. 基础语法

    1. 编码

    2. 标识符

    3. Python保留字（关键字）

      import keyword:Python标准库提供了一个keyword module

      keyword.kwlist:可以输出当前版本的所有保留字

    4. 注释

      单行注释以#开头，多行注释采用三对单引号或者三对双引号

    5. 缩进

      使用缩进来表示代码块。缩进的空格是可变的，默认4空格，使用同一个代码块的语句必须包含相同的缩进空格数

    6. 标准数据类型（六个标准的数据类型）

      1. Number（数字）
      2. String（字符串）
      3. List（列表）
      4. Tuple（元组）
      5. Set（集合）
      6. Dictionary（字典）

      不可变数据：数字、字符串、元组

      可变数据：列表、字典、集合

    7. 字符串

      1. 单引号和双引号使用完全相同
      2. 使用三对引号可以囊括一个多行字符串
      3. 转义符\
      4. 自然字符串，通常在字符串前面加r或R
      5. 允许处理unicode字符串，加前缀u或U
      6. 字符串是不可变的
      7. 有两种索引方式，从左往右以0开始，从右往左以-1开始
      8. 截取的语法格式：变量【头下标：尾下标：步长】

    8. 空行

      函数之间或类的方法之间用空行分隔，表示一段新的代码的开始。类和函数入口之间也用一行空行分隔，以突出函数入口的开始。空行的作用在于分隔两段不同功能或含义的代码，便于日后代码的维护或重构

    9. 等待用户输入

      input（"\n\n按下enter键后退出"）

    10. 同一行显示多条语句

      同一行使用多条语句，语句之间用分号;分隔

    11. 多个语句构成代码组

      缩进相同的一组语句构成一个代码块，也称代码组。

      首行以关键字开始，以冒号：结束，该行之后的一行或多行代码构成代码组

      首行及后面的代码组称为一个子句（clause）

    12. print输出

      默认输出是换行的

    13. import与from…import

      导入相应的模块。

      将整个模块导入：import somemodule

      从某个模块中导入某个函数：from somemodule import somefunction

      从某个模块中导入多个函数：from somemodule import firstfunc，secondfunc，thirdfunc

      将某个模块中的全部函数导入：from somemodule import *

    14. 命令行参数

      python -h查看各参数帮助信息
